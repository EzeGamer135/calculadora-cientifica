Logaritmación

Fórmula General

$$\log_b x = y$$

El logaritmo de un número x en una base b es el exponente y al que se debe elevar la base para obtener dicho número:

$$b^y = x$$

Esta función es útil en diversos campos, incluyendo matemáticas, informática y economía, permitiendo resolver ecuaciones exponenciales y analizar crecimiento exponencial.
